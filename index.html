MEMORY, REGISTERS & CPU
Cache · RAM/ROM · FDE cycle · Virtual memory · Computer types
Section A  5/5 YEARS
Q2e
Name the THREE phases of the instruction cycle and outline how EACH phase operates. [6 marks]
6 marks2017REPEAT›
✓ Full Answer▾
1. FETCH: PC holds next instruction address → copied to MAR → instruction retrieved from RAM to MDR → copied to CIR → PC incremented.

2. DECODE: Control Unit interprets opcode in CIR, generates control signals and identifies operands.

3. EXECUTE: ALU/hardware performs the operation; result stored in register or RAM; cycle repeats.
Q2f
Discuss why a cache is expected to speed up a computer system. Include a definition. [4 marks]
4 marks2017REPEAT›
✓ Full Answer▾
Definition: Cache is a small, very fast memory between the CPU and RAM that stores copies of frequently/recently accessed data.

Exploits temporal locality (reuse of same data) and spatial locality (nearby data). Cache hits retrieve data much faster than RAM access, bridging the CPU–RAM speed gap.
Q2e
With reference to RAM, ROM and hard disks, explain why different types of memory are needed. [6 marks]
6 marks2018REPEAT›
✓ Full Answer▾
RAM: Volatile, fast, holds active programs/data. Lost on power-off.
ROM: Non-volatile, permanent firmware (BIOS). Cannot be modified normally.
Hard Disk: Non-volatile, large capacity, slow. Persistent file storage.

No single technology can simultaneously offer high speed + large capacity + non-volatility + low cost. Each occupies a niche in the memory hierarchy.
Q2a
(i) Define 'cache memory'. (ii) Explain how cache memory increases the efficiency of data retrieval in the CPU. [5 marks]
5 marks2022REPEAT›
✓ Full Answer▾
(i) Cache memory is a small, high-speed memory located between the CPU and main RAM that stores copies of frequently or recently used data and instructions.

(ii) When the CPU needs data, it first checks the cache. If found (cache hit), it retrieves data in nanoseconds rather than the slower RAM access time. This reduces the number of slow main-memory accesses, significantly increasing overall throughput. Cache exploits the principle of locality — programs tend to reuse the same small set of data repeatedly.
Q2b
(i) Name TWO components of the CPU, other than registers. (ii) State the function of EACH. [4 marks]
4 marks2022REPEAT›
✓ Full Answer▾
1. Control Unit (CU): Directs and coordinates all operations in the CPU — fetches instructions, decodes them, and generates control signals to execute them.

2. Arithmetic Logic Unit (ALU): Performs all arithmetic operations (add, subtract) and logical operations (AND, OR, NOT, compare) on data.
Q2c
Describe a scenario where each can be used: (i) Supercomputer, (ii) Server, (iii) Mobile device. [6 marks]
6 marks2022›
✓ Full Answer▾
(i) Supercomputer: Weather forecasting agencies use supercomputers to simulate atmospheric conditions over millions of grid points simultaneously, producing forecasts that would take years on ordinary hardware.

(ii) Server: An e-commerce company uses web servers to handle thousands of simultaneous customer requests, managing product databases and transactions 24/7.

(iii) Mobile device: A student uses a smartphone to take photos, access cloud storage, and video-call classmates while travelling to school.
Q1a
Differentiate between: (i) ROM and RAM, (ii) Surge protector and voltage regulator, (iii) Supercomputer and microcomputer. [6 marks total]
6 marks2023REPEAT›
✓ Full Answer▾
(i) ROM vs RAM: ROM is non-volatile (retains data without power) and read-only in normal operation; used for firmware/BIOS. RAM is volatile (loses data on power-off) and read/write; used for temporary active program storage.

(ii) Surge protector vs voltage regulator: A surge protector shields equipment from sudden voltage spikes (e.g. lightning). A voltage regulator continuously maintains a stable output voltage regardless of fluctuations in the input supply.

(iii) Supercomputer vs microcomputer: A supercomputer is an extremely powerful, expensive machine used for complex scientific calculations (e.g. climate modelling). A microcomputer is a small, affordable personal computer (desktop, laptop) used by individuals for everyday tasks.
Q1b
A gamer wishes to purchase a new computer. State THREE hardware specifications that should be considered and explain why EACH was chosen. [6 marks]
6 marks2023›
✓ Full Answer▾
1. GPU (Graphics Processing Unit): A dedicated GPU (e.g. NVIDIA GeForce) is essential for rendering complex 3D graphics at high frame rates, enabling smooth gameplay and high visual quality.

2. RAM (amount): At least 16GB RAM ensures the game and OS can run simultaneously without slowdowns; games load large assets into memory during play.

3. CPU speed (GHz / core count): A fast multi-core processor handles game logic, AI calculations, and physics simultaneously. Higher clock speeds reduce processing latency.
Q1c
Briefly describe the processes in the fetch-decode-execute instruction cycle. [3 marks]
3 marks2023REPEAT›
✓ Full Answer▾
Fetch: The CPU reads the next instruction from RAM (at the address in the Program Counter) into the CIR; the PC is then incremented.
Decode: The Control Unit interprets the opcode and prepares the necessary control signals.
Execute: The instruction is carried out (e.g. ALU performs arithmetic, data is moved, a branch is taken); results are stored.

Q3a
State how the top-down problem-solving method differs from the bottom-up method. [2 marks]
2 marks2023›
✓ Full Answer▾
Top-down: The problem is broken down from the overall goal into smaller and smaller sub-problems (stepwise refinement), until each sub-problem is simple enough to solve directly.

Bottom-up: Small, reusable components/modules are built and tested first, then combined to form larger parts of the solution, working up to the complete system.
Q3b

4 marks2023›
✓ Full Answer▾
(i) Problem definition: Clearly stating what the problem is — its scope, inputs, outputs and constraints.
(ii) Problem analysis: Breaking the problem into components; understanding relationships and identifying what data is needed.
(iii) Identify and evaluate possible solutions: Listing alternative approaches and comparing them on criteria such as efficiency, feasibility and cost.
(iv) Implementation and review: Coding/building the chosen solution, testing it against requirements, and refining based on results.

int arraySum(int arr[], int size) {
